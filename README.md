# Haunted Hotel Investigator - Prototype 1

### Edward Wang

### Platform: Windows at home | MacOS on Campus

# Question & Hypothesis

Question: Does limiting time during the preparation phase of an investigative game create for more meaningful and heightened play?
Hypothesis: If the preparation scene uses...

-   A hard "Elevator Departure" Countdown
-   Limited Inventory Slots
-   Ambient audio to heighten the mood
    Then players will make faster and more instinctual / emotional choices and be more prone to:
-   Rushing
-   Second-Guessing

# What's Included

This prototype includes:

-   Visual: UI Briefing Panel, Tool Buttons, Loadout Display, Countdown Timer, Result Panel
-   Audio: Looping and Ambient Music
-   Interactions: Clicking buttons to select or deselect tools
-   A scene reloading functionality (R)

# Controls

### Mouse

Left click tool buttons to select and deselect tools (max 3)

### Keyboard

Press R to reload scene
